# Norti Design

**Estúdio de Design Gráfico e UI/UX**

A Norti Design é um estúdio independente especializado em identidade visual, experiência do usuário (UI/UX) e modelagem 3D. Atuamos com design estratégico, prototipagem e soluções visuais que conectam marcas e pessoas.

---

## Estrutura do Projeto

- `index.html` — Página inicial
- `about.html` — Sobre o estúdio
- `projetos.html` — Galeria de projetos
- `css/style.css` — Estilos principais
- `js/scripts.js` — Scripts adicionais (vazio inicialmente)
- `img/` — Imagens de projetos e logotipo
